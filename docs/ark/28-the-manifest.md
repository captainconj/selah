We built the machine because the box it came in had a manifest. Now we have a library of life — 20 proteins — and we are playing them through the machine. This is the packing list. What's in the box, why it's there, and what we expect to find.

## The Experiment

For each protein:

```clojure
(require '[selah.dna :as dna])
(dna/experiment "p53")           ;; or any name from the library
```

This will:
1. Fetch the protein sequence (cached after first fetch)
2. Map each amino acid to its Hebrew letter
3. Slide a 3-letter window across the Hebrew
4. Ask the oracle what it sees at each position
5. Compute word frequencies
6. Save results to `data/dna/`

The priest reads the stones. The oracle ranks the readings. We observe and report.
