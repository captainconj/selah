## Structure

### Part I: The Fold
How we got here. The Torah has 304,850 letters. The number factors. The dimensions have meaning. The center says "guard." The breastplate sits on the heart.

### Part II: The Machine
The breastplate oracle. Four readers. The courtroom. How the letters light up and rearrange. What the machine sees when it reads.

### Part III: The Screw
We built a second machine — one that reads the Torah in every direction the geometry defines. What a fiber is. What a host word is. What direction density means.

### Part IV: What the Screw Found
One chapter per discovery. Each standalone. Each reproducible. Each telling one face of the same story.

### Part V: The Letter
What are we being shown? The Torah is a love letter to the separated heart. Come home.
