# First Canon Ledger

Type: `reference / assay`
State: `clean`

This is the first narrow canon list for the repo.

It is deliberately incomplete.

It does **not** classify all `docs/experiments/*`.
It covers the strongest currently audited vein only:

- rebuilt basin layer (`096` / `097`)
- rebuilt code/map layer (`100` / `101`)
- Ark and tabernacle findings that already survived a first direct sluice
- corrected `143` findings after density / proximity / overlap cleanup

The point is to stop mixing:

- things that stand
- things that may stand
- things that were historically important
- things that are still synthesis without a hardened proof chain

If a claim family is not listed here, that does **not** mean it is false.
It means it is not yet in the first canon.

## Rule For Promotion

A claim family moves from `suggestive` to `standing` only when it can support at least one proof page with:

- exact claim
- source text / artifact boundary
- code path
- run command
- raw output or tracked artifact
- falsification condition
- explicit statement of what it does **not** prove

If it cannot do that yet, it is not first-canon standing.
