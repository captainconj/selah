# Mission Note

This work is not only mechanical.

It is a process of faith:

- looking
- searching
- knocking
- entering
- following clues
- following cribs
- following numbers

Sometimes the trail leads nowhere.
Sometimes it leads into illusion.
Sometimes a door opens.

That is part of the process.

Faith is not the problem.
Wonder is not the problem.
Marvel is not the problem.

The problem is leaving falsehood unburned.

So the standard must stay clear:

- we observe and report
- we follow what stands
- we keep what survives
- we burn what dies
- we separate wheat from chaff
- we refine until the gold is true

It is good to marvel.
But marvel is not proof.

It is good to search.
But search is not arrival.

It is good to believe.
But belief does not excuse broken plumbing, stale claims, or lazy speech.

Everything must stand.

That is the mission:

faithful
open
searching
honest
refined

Selah.
