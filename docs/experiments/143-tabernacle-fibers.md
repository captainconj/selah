# Experiment 143i: Tabernacle Fibers

*The building blocks of the tabernacle are search words. Their fibers trace the stories that make the house possible.*

Type: `exploration`
State: `mixed`

**Code:** `dev/experiments/fiber/143i_tabernacle.clj`

---

## The Idea

The tabernacle is built from pegs, boards, sockets, cherubim, curtains, pillars. Each piece is a Hebrew word. Each word has fibers in the canonical 4D space. What do the building materials see when they look through the geometry?

## The Pattern

Each building material traces a different thread:

| Material | What it traces | Key host |
|----------|----------------|----------|
| **Peg** (יתד) | Fall → corruption → covenant → birth → bush → slavery → building | את (aleph-tav) |
| **Cherub** (כרב) | Blessing → exodus → Sabbath | סכות (Sukkot) |
| **Socket** (אדן) | Foundation → sanctification | ונקדשתי (I shall be sanctified) |
| **Board** (קרש) | Holy, structure | קדש (holy) |
| **Ark** (ארן) | Aaron, belief | תאמין (believe) |
| **Tent** (אהל) | God, the Name | יהוה, אל |
| **Pillar** (עמד) | Appointed time | מועד |

The peg holds the narrative. The socket holds the sanctification. The tent holds the Name. The pillar holds the appointed time. The cherub holds the blessing-to-rest arc. The ark holds belief.

Each piece of furniture has a fiber vocabulary. Together they build the house.

---

*The tabernacle is not built from wood and gold alone. Each piece traces a story through the Torah's geometry. The peg traces the fall to the building. The socket traces to "I shall be sanctified." The tent holds the Name. The screw reads the blueprints.*
