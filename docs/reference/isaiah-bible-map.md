# Isaiah–Bible Map

*66 chapters. 66 books. The Bible in miniature.*

Isaiah has 66 chapters. The Bible has 66 books. Isaiah splits 39/27 (judgment/comfort). The Bible splits 39/27 (Old Testament/New Testament). The parallel is ancient — noticed by many readers across centuries.

This table maps each Isaiah chapter to its corresponding Bible book, with letter counts and verse counts for the Isaiah side.
